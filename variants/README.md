# Landing Variants

Five distinct landing page variants for the same product.

- 01-atlas: editorial, refined, atlas motif
- 02-tide: coastal calm, soft gradients
- 03-studio: modern studio, grid + sharp typography
- 04-terrace: warm organic, sunlit palette
- 05-pulse: data-led, high-contrast tech vibe

Open any variant by serving the folder or opening `index.html` in a browser.
